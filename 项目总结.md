# 好又来西餐厅 — 微信小程序项目总结

## 项目概述

一个完整的餐厅外卖小程序，包含菜单浏览、购物车、消息通知、个人中心等功能。后端基于 Express + Vue 3，前端为微信小程序。

---

## 技术栈

| 层级 | 技术 |
|---|---|
| 前端 | 微信小程序原生框架 |
| 后端 | Express 5 (Node.js) |
| 管理后台 | Vue 3 + Pinia + Element Plus |
| 图片处理 | sharp |
| 包管理 | npm |

---

## API 接口

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/menu` | 获取菜单（分类+菜品） |
| GET | `/api/categories` | 获取分类 |
| GET | `/api/notices` | 获取通知 |
| POST | `/api/notices/read-all` | 全部已读 |
| GET | `/api/user` | 获取用户信息 |
| GET | `/api/cart` | 获取购物车 |
| POST | `/api/cart/add` | 添加购物车 |
| POST | `/api/cart/update` | 更新数量 |
| POST | `/api/cart/remove` | 删除商品 |
| POST | `/api/cart/clear` | 清空购物车 |

## 功能模块

### 1. 首页
品牌形象页，背景图 + 进入店铺按钮。

### 2. 菜单
- 左侧分类导航 + 右侧菜品列表
- 点击左侧分类 → 右侧自动滚动到对应区域
- 右侧滚动 → 左侧高亮联动
- 菜品卡片展示：图片、名称、描述、价格
- 一键加入购物车

### 3. 购物车
- 商品列表（图片、名称、单价、数量）
- 数量增减、单件删除、一键清空
- 底部合计 + 结算按钮

### 4. 消息通知
- 分类标签：全部 / 优惠 / 订单 / 系统 / 公告
- 卡片式布局，展开查看详情
- 展开自动标记已读，一键全部已读
- 未读红点提示

### 5. 个人中心
- 用户头像、昵称
- 统计数据：订单数、待处理、优惠券、积分
- 功能入口：订单、地址、收藏、设置等

## 启动方式

```bash
# 启动后端
cd backend
npm install
npm run api
# 输出: API 服务器已启动：http://localhost:3000

# 启动管理后台（可选）
npm run dev
```

小程序：微信开发者工具打开 `miniprogram/` 文件夹，修改 `utils/api.js` 中的 `BASE_URL` 为后端 IP 地址。

---

## 图片资源

所有餐品图片统一压缩至 400px 宽，png 压缩至约 50-100KB。小程序总包大小控制在 2MB 以内。

图片命名使用英文：`signature_beef_burger.png`、`spicy_chicken_burger.png` 等。
