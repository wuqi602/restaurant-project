# 项目介绍

本项目为 **小程序开发实训课程设计**——好又来西餐厅微信点餐小程序。

## 说明

- 本项目为课程设计作品，结构简单，适合初学者参考学习
- 仅包含基础框架，实现了核心的点单流程功能
- 适合作为微信小程序开发、Node.js 后端开发的课程设计参考

## 适用对象

需要做微信小程序课程设计、毕业设计的朋友可以参考使用。

## 功能范围

- 首页 / 菜单浏览 / 商品详情
- 购物车管理 / 结算下单
- 订单查看 / 消息通知 / 个人中心
- 基础 Express 后端 API（内存模拟数据，无需配置数据库）

## 注意事项

- 后端使用内存数组模拟数据，重启后数据重置
- 未集成真实支付、用户登录、数据库等功能
- 管理后台（Vue）为基础框架搭建，管理功能待完善

## 许可

仅供学习参考，请勿直接作为商业项目使用。
